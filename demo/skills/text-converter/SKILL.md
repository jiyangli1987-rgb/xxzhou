# 文本格式转换器

提供简单的文本格式转换功能，专注于将Markdown格式转换为HTML格式。

## 支持的转换

### Markdown → HTML

将常见的Markdown语法转换为对应的HTML标签：

#### 标题转换
- `# 标题一` → `<h1>标题一</h1>`
- `## 标题二` → `<h2>标题二</h2>`
- `### 标题三` → `<h3>标题三</h3>`

#### 文本格式
- `**粗体**` → `<strong>粗体</strong>`
- `*斜体*` → `<em>斜体</em>`
- `` `代码` `` → `<code>代码</code>`

#### 列表转换
- `- 项目1` → `<ul><li>项目1</li></ul>`
- `1. 步骤1` → `<ol><li>步骤1</li></ol>`

#### 链接和图片
- `[链接文本](URL)` → `<a href="URL">链接文本</a>`
- `![替代文本](图片URL)` → `<img alt="替代文本" src="图片URL">`

## 使用方法

1. 识别源格式（通常是Markdown）
2. 应用对应的转换规则
3. 生成目标格式的文本
4. 保持内容的完整性和可读性

## 注意事项

- 保持原始文本的段落结构
- 正确嵌套HTML标签
- 处理特殊字符的转义
- 确保生成的HTML语法正确
