# 文本格式转换器 Skill

这是一个简单的文本格式转换技能示例，演示Claude Code中skill的基本结构和使用方法。

## 功能概述

将Markdown格式的文本转换为HTML格式，包含基本的格式转换规则。

## 文件结构

```
text-converter/
├── SKILL.md              # 技能主文档
├── README.md             # 使用说明
└── examples/             # 示例文件
    ├── sample.md         # 示例Markdown文档
    └── sample.html       # 对应的HTML转换结果
```

## 触发方式

当用户说以下类型的句子时，会触发此技能：
- "把这段markdown转成HTML"
- "转换为HTML格式"
- "markdown转换"

## 学习要点

### Skill开发要点

1. **单一职责**：这个skill只做一件事情 - 文本格式转换
2. **清晰的触发条件**：description明确列出了触发短语
3. **简单的文件结构**：只有一个SKILL.md文件
4. **实际示例**：提供具体的转换示例

### 与复杂Skill的对比

| 特性 | 简单Skill | 复杂Skill（如文件重命名） |
|------|-----------|---------------------------|
| 文件数量 | 1-2个 | 多个目录和文件 |
| 功能范围 | 单一 | 多功能 |
| 学习曲线 | 低 | 高 |
| 演示时间 | 5分钟 | 30分钟+ |

这个简单的skill非常适合作为教学演示，帮助初学者快速理解skill的基本概念和工作原理。
